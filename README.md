## 4단계 요구사항

* 노선에 요금 필드를 추가
* 요금을 포장 객체를 이용하여 작성 (단위 테스트 작성) -> 작성 까지만
* 기존 인수 테스트 수정
* 지하철 이용 요금이 포함된 인수테스트 작성
    * 노선 별 추가 요금 정책 개발
        * 추가 요금 로직 개발 (단위 테스트 작성)
        * 구간 중 추가요금 노선에 관한 로직 적용
        * 테스트를 통하여 요금 로직 확인
    * 로그인 사용자의 경우 연령별 요금 할인 적용
        * 청소년, 어린이 체크 로직 (인증 테스트) 구현
        * 구현 시 인터페이스를 이용하여 로직을 작성 (추후 계획 변동 요지)
        * 테스트를 통하여 요금 로직 확인
* 복잡한 로직 단순화 리팩토링
    * 요금 측정
    * 로그인하지 않은 유저 검증 방법
    * Fare 리팩토링
    * 테스트케이스 정리
* 청소년, 어린이 요금제 인수 테스트 작성
